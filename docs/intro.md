# Introducción

Bienvenido a la documentación de la API SIDIS v4.

Esta documentación te permitirá:

- **Leer la documentación** completa de todos los endpoints
- **Probar la API** directamente desde la documentación
- **Configurar tu entorno** de testing
- **Copiar comandos cURL** para usar en tu terminal

## Características Principales

### Documentación Interactiva
Cada endpoint puede ser probado directamente desde esta documentación. Solo necesitas:

1. Configurar la URL base de tu API
2. Añadir tu token JWT si es necesario
3. Llenar los parámetros requeridos
4. Hacer click en "Execute"

### Soporte Completo para CORS
La documentación funciona desde el navegador gracias a CORS habilitado en tu API.

### Exportación a cURL
Cada petición puede ser copiada como comando cURL para usar en terminal o scripts.

## Comenzar

Para comenzar a usar la documentación interactiva, ve a la sección [API Interactive](./api-interactive) y configura tu URL base y token de autenticación.
